# ARKCARDS DATA MANAGER

### Introduction
	- The purpose of the program is to allow access to the MySQL database for the ARKARDS system.
	- The intent of this program is to give an easy way of viewing, adding, and removing clients from the database.

### Installation 
	- Install the latest version of MySQL, ideally to be installed onto a dedicated sever for maximum performance/security. However for our case use development intall.
	- Ensure the latest version of python is installed on the PC
	- Use cmd to install the following modules with this command (WINDOWS): pip install
